# 简介

基于 react dumi 的一个 UI 框架

## 使用

安装库及相关依赖包

```bash
# 安装 ui 包
$ npm install lei-view
```

导入样式，在入口文件（layout.tsx/index.tsx）

```js
import 'lei-view/dist/umd/lei-view.min.css'
```
